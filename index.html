Como Funciona o Aplicativo

Este é um gerador de imagens de IA que utiliza a API da OpenRouter. Para começar, siga estes passos:

  1. **Descreva sua ideia:** No campo de texto, escreva o que você deseja criar. Seja o mais detalhado possível para obter melhores resultados.
  2. **Escolha um estilo:** Selecione um dos estilos pré-definidos (como "Arte Digital" ou "Fotorrealista"). Isso adicionará palavras-chave ao seu prompt.
  3. **Obtenha sua chave:** A chave de API é necessária para usar o gerador. Você pode obtê-la clicando em "Adquirir Chave" ou "Suporte" no menu superior.
  4. **Insira sua chave:** Cole a chave no campo "Sua Chave API" e clique em "Salvar". A chave será salva no seu navegador por 30 dias.
  5. **Gere sua imagem:** Clique no botão "Gerar Imagem". Aguarde alguns instantes e sua criação aparecerá na tela!

Para qualquer dúvida, entre em contato via WhatsApp no item "Suporte" do menu.
